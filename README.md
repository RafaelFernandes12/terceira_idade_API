# terceira_idade_API

## Como executar

#### **1. Clone o projeto para sua maquina local:**

	git clone https://github.com/RafaelFernandes12/terceira_idade_API.git
	
#### **2. Acesse a pasta**
	
	cd terceira_idade_API
	
#### **3. Rode localmente usando o docker**
	
	docker compose up

Eu sei que subi o .env, depois eu tiro!
